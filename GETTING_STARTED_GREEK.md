# Ξεκινώντας — AI Brain for EU Lawyers · v0.1

**🙏 Καλώς ήρθατε, δικηγόρε — η AI Brain σας ξεκινά εδώ.**

## Τι είναι;

Το AI Brain EU είναι ένας **βοηθός διαχείρισης δικηγορικού γραφείου** για ανεξάρτητους δικηγόρους στην ΕΕ. Λειτουργεί εξ ολοκλήρου στον υπολογιστή σας — χωρίς cloud, χωρίς κλειδί API, χωρίς τηλεμετρία. Οι φάκελοι των υποθέσεών σας δεν εγκαταλείπουν ποτέ τον υπολογιστή σας.

## Εγκατάσταση (30 δευτερόλεπτα)

```bash
cd ~/Desktop
git clone https://github.com/Wolfgangrush/ai-law-firm-eu.git
cd ai-law-firm-eu
pip install -e .
```

## Εκκινήστε το γραφείο σας (2 λεπτά)

```bash
mkdir ~/to-grafeio-mou-eu
ailawfirm-eu init ~/to-grafeio-mou-eu
ailawfirm-eu mine ~/to-grafeio-mou-eu
ailawfirm-eu search "Βρυξέλλες I-bis εκτέλεση"
```

## 7 ειδικοί πράκτορες

| Agent | Λειτουργία |
|---|---|
| `matter_agent` | Παρακολούθηση υποθέσεων |
| `citation_agent` | Επικύρωση παραπομπών ECLI / CELEX |
| `court_agent` | Πληροφορίες δικαστηρίων (ΔΕΕ, εθνικά) |
| `drafting_agent` | Βοήθεια στη σύνταξη (ενήμερη για τη διαδικασία της ΕΕ) |
| `deadline_agent` | Έλεγχος προθεσμιών παραγραφής |
| `calendar_agent` | Συγχρονισμός ημερολογίου ICS (ζώνη ώρας Ευρώπη/Βρυξέλλες) |
| `compliance_agent` | AI Act Παράρτημα III + GDPR + Ξέπλυμα + CCBE |

## ⚠️ Προειδοποίηση για την AI Act

Ο **Ευρωπαϊκός Κανονισμός για την ΤΝ (2024/1689)** κατατάσσει τα συστήματα ΤΝ που χρησιμοποιούνται « στη διοίκηση της δικαιοσύνης » ως **ΥΨΗΛΟΥ ΚΙΝΔΥΝΟΥ** (Παράρτημα III, σημείο 8). Αυτό το εργαλείο είναι λογισμικό διαχείρισης γραφείου (Ελάχιστος Κίνδυνος). ΔΕΝ εκτελεί δικαστική συλλογιστική ή πρόβλεψη αποφάσεων.
