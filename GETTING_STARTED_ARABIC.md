# ابدأ — AI Brain for EU Lawyers · v0.1

**🙏 أهلاً وسهلاً، يا محامي — تبدأ AI Brain الخاصة بك من هنا.**

## ما هو؟

AI Brain EU هو **مساعد لإدارة المكتب القانوني** للمحامين المستقلين في الاتحاد الأوروبي. يعمل بالكامل على جهازك — لا سحابة، لا مفتاح API، لا تتبع عن بعد. ملفات قضاياك لا تغادر جهاز الكمبيوتر الخاص بك أبداً.

## التثبيت (30 ثانية)

```bash
cd ~/Desktop
git clone https://github.com/Wolfgangrush/ai-law-firm-eu.git
cd ai-law-firm-eu
pip install -e .
```

## ابدأ مكتبك (دقيقتان)

```bash
mkdir ~/maktabi-eu
ailawfirm-eu init ~/maktabi-eu
ailawfirm-eu mine ~/maktabi-eu
ailawfirm-eu search "بروكسل I-bis تنفيذ"
```

## 7 وكلاء متخصصون

| Agent | الوظيفة |
|---|---|
| `matter_agent` | متابعة القضايا |
| `citation_agent` | التحقق من صحة الاستشهادات ECLI / CELEX |
| `court_agent` | معلومات المحاكم (محكمة العدل الأوروبية، المحاكم الوطنية) |
| `drafting_agent` | المساعدة في الصياغة (واعية بإجراءات الاتحاد الأوروبي) |
| `deadline_agent` | التحقق من مواعيد التقادم |
| `calendar_agent` | مزامنة التقويم ICS (المنطقة الزمنية أوروبا/بروكسل) |
| `compliance_agent` | قانون الذكاء الاصطناعي الملحق الثالث + اللائحة العامة لحماية البيانات + مكافحة غسل الأموال + CCBE |

## ⚠️ تنبيه قانون الذكاء الاصطناعي

تصنف **اللائحة الأوروبية للذكاء الاصطناعي (2024/1689)** أنظمة الذكاء الاصطناعي المستخدمة « في إقامة العدل » على أنها **عالية المخاطر** (الملحق الثالث، النقطة 8). هذه الأداة هي برنامج لإدارة المكاتب (مخاطر دنيا). لا تقوم بالاستدلال القضائي أو التنبؤ بالأحكام.

## الخصوصية

كل شيء محلي. متوافقة مع اللائحة العامة لحماية البيانات بطبيعة التصميم. لا تغادر أي بيانات جهازك.
